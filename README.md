
# 🏴‍☠️ OverTheWire: Bandit Walkthrough

A HTML and MD walkthrough for the [OverTheWire: Bandit](https://overthewire.org/wargames/bandit/) wargame (Levels 0–33). 



## 🚀 Usage

Since it's a single webpage, no server or installation is required.

Simply download the file and open it in any modern web browser:

```bash
# Open from terminal
xdg-open bandit-walkthrough.html  # Linux
open bandit-walkthrough.html      # macOS
start bandit-walkthrough.html     # Windows
```

*Alternatively, just double-click `bandit-walkthrough.html` from your file manager.*

Those who are using the .md file open it in some .md viewer

## ⚠️ Disclaimer

The primary objective of OverTheWire is to *learn* security concepts and Linux command-line skills by figuring things out yourself. Use this walkthrough as a reference or a hint guide if you are truly stuck, but try your best to solve the challenges independently first!
